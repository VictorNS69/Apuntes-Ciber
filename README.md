## Requisitos
Instalar **_LaTex_**
```bash
sudo apt install texlive-full
```
Instalar **_rubber_**
```bash
sudo apt install rubber
```

## Compilar los LaTex
Ir al directorio donde se encuentra el archivo que deseas compilar y ejecutar el _Makefile_.
```bash
cd DIRECTORIO_X
make
```
